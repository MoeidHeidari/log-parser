    Overview

    Log parser takes a log input file and tries to parse it to extract useful information. It also respects General Data Protection Regulation (GDPR)

    You can see take a look at the Full documentation here

Provided Commands

cli

parses a log file

Options:
  -h, --help       Show help                                           [boolean]
  -l, --log-level  log-level (ex: "error,[error,debug,warn,info]")
                                                     [string] [default: "error"]
  -o, --output     output log file (ex: "output.log")
                                     [string] [required] [default: "output.log"]
  -i, --input      input log file (ex: "input.log")          [string] [required]
  -v, --version    Show version number                                 [boolean]
